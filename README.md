# poplar

## 简介
`poplar`是永辉云创C端H5页面所使用的UI组件库

## 开发
```bash
// 安装依赖
npm install

// 开始开发
npm start

// 编译UI库
npm run build:lib

// 发布lib到npm
npm publish

// 打包Example网站
npm run build
```
